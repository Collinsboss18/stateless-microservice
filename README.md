## Nodejs stateless microservice

### Functionalities

* Authentication
* JSON patching
* Image Thumbnail Generation

### Setup

> Usage instruction

#### Start Project

1. Clone project
2. `cd` clone
3. Rename *local.env* to *.env* and add configurations
4. `npm install`.
5. run `npm start`

> With Docker
4. run `docker build .`


#### For test

* run `npm test`.
